# 贝壳社区    
基于SpringBoot开发的编程交流论坛        

##部署     
- Git      
- JDK     
- Maven      
- MySQL      

##功能      
- 首页(手写分页功能)
- 问题发布修改
- 一级评论二级评论
- 搜索
- 消息通知
- 热门话题(使用TopN实现热度排序)
